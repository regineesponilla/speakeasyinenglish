# SpeakEasy in English 🍀

Your personal English confidence coach — built for Filipino professionals in Ireland.

## Setup

1. Clone this repo
2. Install dependencies:
   ```
   npm install
   ```
3. Create your `.env` file:
   ```
   cp .env.example .env
   ```
4. Add your Anthropic API key to `.env`:
   ```
   VITE_ANTHROPIC_API_KEY=your_key_here
   ```
5. Run locally:
   ```
   npm run dev
   ```

## Deploy to GitHub Pages

```
npm run deploy
```

This builds the app and pushes to your `gh-pages` branch automatically.
